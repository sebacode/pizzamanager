## PizzaManager

Web application to manage pizzas, made with the Laravel framework and using MYSQL database.

## Features

- Create Pizza.
- Modify Pizzas.
- Assign ingredients to a pizza.
- Sort ingredients according to user.
- Delete Pizzas.
- Create Ingredients.
- Modify Ingredients.
- Delete Ingredients.
- ..
